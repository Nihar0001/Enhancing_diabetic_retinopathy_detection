## 📂 Step-by-Step: Downloading the Data Folder

### Step 1: Open the Google Drive Link
Visit: https://drive.google.com/drive/folders/1lKMGO2NrZ67wH5LJkvFxRpF75I89CgZt?usp=drive_link

### Step 2: Download the Entire Folder
- Click the folder name to open it
- Click the **"Download all"** button (or right-click the folder → **Download**)
- Google Drive will compress everything into a `.zip` file

### Step 3: Extract the Files
```bash
# On Linux/macOS
unzip <downloaded_file>.zip -d /path/to/project/data/

# On Windows (PowerShell)
Expand-Archive -Path <downloaded_file>.zip -DestinationPath .\data\
```

### Step 4: Verify the Folder Structure
After extracting, your `data/` folder should look like:
```
data/
├── train_images/        # Folder containing retina training images
├── test_images/         # Folder containing retina test images
├── train.csv            # Training labels (image filename → DR grade)
├── test.csv             # Test labels
├── X_features.npy       # Extracted feature matrix
├── X_train.npy          # Training feature matrix
├── X_test.npy           # Test feature matrix
├── X_train_scaled.npy   # Scaled training features
├── X_test_scaled.npy    # Scaled test features
├── y_train.npy          # Training labels array
├── y_test.npy           # Test labels array
└── y_labels.npy         # All labels array
```

## 🛠️ Troubleshooting

### "Access Denied" on Google Drive
- Make sure you are signed into a Google account
- If the link has expired, contact the project owner for updated links

### Files extracted to wrong location
- Ensure the extracted files are directly under `data/` (not in a subdirectory like `data/data/`)
- Check: `ls data/` should list `train.csv`, `train_images/`, etc.

### "Data files not found" when running scripts
- Confirm `config.py` shows the correct `Project Root`
- Confirm `data/train.csv` and `data/train_images/` exist
- Re-run `python config.py` to see current paths

### Large download size / slow download
- Use a wired connection for reliability
- Consider downloading individual files instead of the full folder if only some are needed
